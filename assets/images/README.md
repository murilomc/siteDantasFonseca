# Imagens do site

As imagens utilizadas pela página principal e pelo artigo estão armazenadas
localmente nesta pasta. Os HTML usam caminhos relativos, enquanto Open Graph e
JSON-LD usam os endereços públicos esperados em:

`https://dantasfonseca.adv.br/assets/images/`

O arquivo `image-manifest.json` relaciona cada arquivo à sua utilização.
